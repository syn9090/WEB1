JavaScript란 무엇인가?

JavaScript , [6] often abbreviated as JS, is a high-level, interpreted programming language. It is a language which is also characterized as dynamic, weakly typed, prototype-based and multi-paradigm. Alongside HTML and CSS, JavaScript is one of the three core technologies of the World Wide Web. [7] JavaScript enable interactive web pages and thus is an essential part of web applications, The vast majority of webside use it, [8] and all major web browsers have a dedicated JavaScript engin to execute it.
