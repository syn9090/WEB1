Hypertext Markup Language (HTML) is the starndard markuplanguage for creating webpages and web applications. Web browsers received HTML documents from a web server or from local storage and render them into mulitmediia web pages. HTML describes the structure of a web page semantically standard

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such a interactive forms, may be embedded into the rendered page. It provides a means to create structured documents by denoting structural semantics for text such as headings, parapgaphs, list, links, quotes and other items, HTML elements are delineated by tags, writteen using angle brackets.
